# dotnet-clean-architecture-api
# .NET Clean Architecture REST API

## Overview
Production-ready REST API designed following Clean Architecture principles.
The project focuses on scalability, security, and maintainability.

## Architecture
- .NET 8
- Clean Architecture
- SQL Server
- JWT Authentication
- Docker

## Key Features
- RESTful endpoints
- JWT authentication and role-based authorization
- SQL Server with optimized queries
- Global error handling
- Logging and health checks

## How to Run
Docker and local setup instructions will be added.

## Technical Decisions
This project applies Clean Architecture to ensure separation of concerns,
testability, and long-term maintainability.
